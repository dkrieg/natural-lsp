# natural-lsp — VS Code extension

Companion VS Code extension (TypeScript) that launches the `natural-lsp` server
when a Natural source file is opened. Zero-config when the server is on `PATH`;
overridable via the `naturalLsp.serverPath` setting.

See the root [README](../../README.md) (Editor setup → VS Code) and
`docs/plans/natural-lsp-prd.md` (FR-44).
